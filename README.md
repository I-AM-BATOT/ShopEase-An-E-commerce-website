# 🛒 E-Commerce Website (Node.js)

## 📌 Project Description

This is a basic e-commerce web application built using Node.js. It allows users to browse products, view product details, add items to a shopping cart, and place orders. The project demonstrates full-stack development with user authentication and order management.

---

## 🚀 Features

* User Registration & Login
* Product Listings
* Product Details Page
* Shopping Cart Functionality
* Order Processing System
* Secure Backend APIs

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js with Express.js
* **Database:** MongoDB *(update if you used something else)*

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/ecommerce-nodejs.git

# Navigate into the project folder
cd ecommerce-nodejs

# Install dependencies
npm install

# Start the server
npm start
```

### 🔑 Environment Variables

Create a `.env` file in the root directory and add:

```
PORT=5000
MONGO_URI=your_database_connection_string
JWT_SECRET=your_secret_key
```
